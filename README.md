<h1 align="center">IMovie</h1>

<p align="center">
IMovie is a sample Android project using <a href="http://www.omdbapi.com/">OMDB</a> API based on MVVM architecture.
</p>

## Features
* Kotlin
* MVVM architecture
* RxJava
* Retrofit
* Room
* Dependency injection(Dagger)
* Other: LiveData, Glide, ViewBinding, Lottie
